# TSTyche Starter Project

This is the TSTyche starter project. You can try it online on StackBlitz:

[![Open in StackBlitz][starter-badge]][starter-url]

## How to Use

The example test files are located within the `examples` directory.

Try out the following commands:

```shell
# Run all tests
npx tstyche
```

```shell
# Only run the matching test file
npx tstyche loading
```

```shell
# Run all tests in watch mode
npx tstyche --watch
```
